# Prompt Library

A web app for managing a personal library of AI prompts. Built in Hebrew with a
right-to-left (RTL) layout and a dark theme. Everything runs in the browser — no
backend needed.

## Features

- Add, edit and delete prompts
- Search and filter your collection
- Dark, RTL-friendly design
- Export / import the whole library as JSON

## Run it

It's a static site — just open `index.html` in your browser.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Page structure |
| `styles.css` | Styling (dark theme, RTL) |
| `app.js` | App logic |
| `prompts.json` | Sample prompt data |
